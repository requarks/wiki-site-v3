# Blocks - Media Player

A basic media player

**Element:** `block-media-player`

## Properties

| Parameter | Description | Default | Required |
|---|---|:---:|:---:|
| `src` | Absolute URL to a supported video file. |  | *Yes* |

## Usage

### Markdown

```markdown
::block-media-player{src="https://example.com/video.mp4"}
::
```
